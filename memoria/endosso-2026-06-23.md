## Risco operacional Supabase

O fallback para JSON acontece quando as variaveis Supabase nao existem.

Se `NEXT_PUBLIC_SUPABASE_URL` e `SUPABASE_SERVICE_ROLE_KEY` existem, o app tenta Supabase. Se a rede ou credencial falhar, o erro sobe como `Supabase read error` em vez de fazer fallback silencioso para JSON.

Isso pode ser desejavel em producao, mas durante desenvolvimento local pode confundir. Para forcar JSON local, remover ou esvaziar as variaveis Supabase em `.env.local`.
